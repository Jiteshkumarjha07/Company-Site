Cursor Color Reference

The cursor ring changes color depending on what you're hovering:

White
Default — text, nav, general content
Purple
CTA Buttons (.btn-cta)
Amber
Team Cards (.team-card, [data-magnetic])
Teal
Canvas Areas (.hero-canvas, .stats-canvas)

Troubleshooting

Cursor not appearing?
  • Check DevTools → Elements → Look for <canvas id="custom-cursor-canvas">
  • Check DevTools → Console for any JavaScript errors
  • Verify cursor: none; is applied globally in CSS
Cursor not changing colors on hover?
  • Verify elements have correct classes: .btn-cta, .team-card, etc.
  • Check that elements have data-cta or data-magnetic attributes
  • Inspect elements in DevTools to confirm classes are present
